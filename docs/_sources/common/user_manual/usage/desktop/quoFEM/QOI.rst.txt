QoI: Quantaties of Interest
===========================

The QoI panel is where the user specifies the variable names associated with the FEM results. For each result quantatity that the users application generates and places in the results.out file, the user must provide a variable name. A new response quantatity is added by the user slecting the ''Add'' button. Quantaties can be removed by selecting it, this is done by clicking in the circle befor the variable name, and theb selecting the ''remove'' button.

In figure shown the user has added two response quantiaites, **Node_2_Disp** and **Node_3_Disp**.

.. note::

   There can be no spaces in variable names, underescores are permitted

   Names cannot be duplicated
